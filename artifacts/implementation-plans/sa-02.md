# Plan for Implementing Story: sa-02

Story: `artifacts/stories/sa-02-intersect-select-friends.md`

## Tasks
1. sa-02-t1
  - Description: Set up Vitest (`npm test` script, config for TS + path alias).
  - Acceptance Criteria:
    - `npm test` runs and passes with a trivial smoke test.
  - Status: Completed

2. sa-02-t2
  - Description: Generic TTL cache in `src/lib/cache.ts` behind one interface (Map-based impl).
  - Acceptance Criteria:
    - Interface: `get(key)`, `set(key, value, ttlMs)` (or equivalent).
    - Expired entries not returned.
    - Unit tests: hit, miss, expiry (fake timers).
  - Status: Completed

3. sa-02-t3
  - Description: Steam Web API client in `src/lib/steam/` — key from `STEAM_API_KEY`, ResolveVanityURL, GetFriendList, GetPlayerSummaries (batch 100), 5-min cache via `lib/cache`.
  - Acceptance Criteria:
    - Server-only module; key never sent to client.
    - SteamID64 validation (17-digit) + vanity detection helper.
    - Friend mapping: { steamId, name, avatarUrl, isPrivate } (private = communityvisibilitystate != 3).
    - Distinct error for private/inaccessible own friend list.
    - Unit tests: validation, mapping, privacy flags, batching; fetch mocked.
  - Status: Not Started

4. sa-02-t4
  - Description: Server Functions in `src/app/steam-intersect/actions.ts` ('use server'): resolve identity + fetch friend list (self included as entry).
  - Acceptance Criteria:
    - Input vanity URL or SteamID64 resolved to SteamID64.
    - Returns friends + self with name/avatar/isPrivate, or typed error (not-found, private-list, api-failure).
  - Status: Not Started

5. sa-02-t5
  - Description: Replace stub `/steam-intersect` page with client UI: SteamID form, error + retry, friend grid with selection.
  - Acceptance Criteria:
    - Form accepts vanity or SteamID64; on error shows clear message, input retained.
    - Friend list shows avatar + name; self in list, selectable like others.
    - Private friends badged, not selectable.
    - Max 10 selected enforced with feedback; Proceed disabled until >= 2.
    - Proceed navigates to `/steam-intersect/results?ids=<csv>`.
    - Uses theme tokens; usable on mobile + desktop widths.
  - Status: Not Started

6. sa-02-t6
  - Description: Verify story end-to-end: build, lint, all tests, manual run with real STEAM_API_KEY.
  - Acceptance Criteria:
    - `npm run build`, `npm run lint`, `npm test` pass.
    - Manual: vanity + SteamID64 resolve, invalid input errors, selection limits work.
  - Status: Not Started

## Excludes
- Results page / game intersection (sa-03).
- Persistent cache (Redis etc.) — interface only.
- Steam OpenID login; user types ID manually.
- Rate limiting beyond 5-min cache.
