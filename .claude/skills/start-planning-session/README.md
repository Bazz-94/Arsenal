# Start Planning Session

Start a structured planning session to break down a feature or task into stories.

## What it does

Guides you through providing context and requirements for a new feature, then breaks it down into stories ready for implementation.

## How to use

Type `/start-planning-session` in the chat.

The skill will:
1. Ask for a brief description of the feature or task.
2. Ask for context: constraints, requirements, dependencies, acceptance criteria.
3. Once you confirm all context is provided, write stories based on your input.

No technical details needed — focus on what users need, not how it will be built.
